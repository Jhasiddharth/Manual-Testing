# Login Module Testing Project

## Project Description

This project demonstrates end-to-end manual testing of a Login module.

## Modules Covered

- Login
- Forgot Password
- Validation Messages

## Testing Scope

- Functional Testing
- Negative Testing
- Input Validation Testing

## Deliverables

- Test Cases
- Bug Reports
- Test Summary Report

## Outcome

The Login module was tested against planned scenarios and defects were documented where applicable.

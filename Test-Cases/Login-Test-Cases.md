# Login Test Cases

| Test Case ID | Test Scenario                          | Expected Result                                   |
| ------------ | -------------------------------------- | ------------------------------------------------- |
| TC_LOGIN_001 | Login with valid username and password | User should be logged in successfully             |
| TC_LOGIN_002 | Login with invalid password            | Error message should be displayed                 |
| TC_LOGIN_003 | Login with invalid username            | Error message should be displayed                 |
| TC_LOGIN_004 | Login with blank username and password | Validation message should be displayed            |
| TC_LOGIN_005 | Verify Forgot Password link            | User should be redirected to Forgot Password page |

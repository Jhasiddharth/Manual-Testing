# BR_001 - Incorrect Error Message on Login Failure

## Summary

Application displays an unclear error message when the user enters an invalid password.

## Module

Login

## Steps to Reproduce

1. Open the Login page.
2. Enter a valid username.
3. Enter an invalid password.
4. Click the Login button.

## Expected Result

A clear message such as "Invalid Username or Password" should be displayed.

## Actual Result

A generic error message is displayed that does not clearly indicate the cause of the failure.

## Severity

Medium

## Priority

Medium

## Status

Open
